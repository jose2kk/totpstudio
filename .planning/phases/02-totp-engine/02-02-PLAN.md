<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input -> secret field | User types arbitrary string that becomes input to otplib generate() |
| otplib generate() -> state | Async crypto operation result stored in React state |
| State -> clipboard | Secret and TOTP code copied to system clipboard via browser API |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-03 | Information Disclosure | Secret in React state | mitigate | useState only — no localStorage/sessionStorage/cookies; state cleared on page unload; secret masked by default (password input type per D-01) |
