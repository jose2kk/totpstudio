## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (e.g. Sun/Moon icon internal spacing) |
| sm | 8px | Compact element spacing (e.g. header item gaps) |
| md | 16px | Default element spacing (e.g. horizontal page padding `px-4`) |
| lg | 24px | Section padding (e.g. card internal padding) |
| xl | 32px | Layout gaps (e.g. vertical page padding `py-8`) |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Header height: 56px (h-14) — Tailwind standard for slim nav bars, source: RESEARCH.md Pattern 6

## Color

All values are shadcn/ui zinc CSS variables — both light and dark modes. Source: CONTEXT.md D-01, D-02.

| Role | Light Mode Value | Dark Mode Value | Usage |
|------|-----------------|-----------------|-------|
| Dominant (60%) | `--background`: hsl(0 0% 100%) — white | `--background`: hsl(240 10% 3.9%) — near-black | Page background, full bleed surfaces |
| Secondary (30%) | `--card`: hsl(0 0% 100%) with `border` at hsl(240 5.9% 90%) | `--card`: hsl(240 10% 3.9%) with `border` at hsl(240 3.7% 15.9%) | Card component, header bar background |
| Accent (10%) | `--primary`: hsl(240 5.9% 10%) — near-black | `--primary`: hsl(0 0% 98%) — near-white | Theme toggle active state only |
| Destructive | `--destructive`: hsl(0 84.2% 60.2%) | `--destructive`: hsl(0 62.8% 30.6%) | Destructive actions only — not used in Phase 1 |

Accent reserved for: theme toggle button active/hover state. No other elements in Phase 1 use primary accent color.

Color system is entirely generated by `shadcn@latest init` via CSS variables in `globals.css`. Do NOT hand-author hex values — use CSS variable references (`bg-background`, `text-foreground`, `bg-card`, `text-primary`, etc.).

Dark mode variant: `@custom-variant dark (&:is(.dark *))` in `globals.css`. Applied by `next-themes` writing `class="dark"` on `<html>`. Source: RESEARCH.md Pattern 4.

## Interaction Contract

### Theme Toggle
- **Trigger:** Click sun/moon icon button in header top-right
- **States:** Light (sun icon visible, moon hidden) / Dark (moon icon visible, sun hidden)
- **Persistence:** next-themes handles via cookie — no custom storage code
- **Initial state:** System default (`defaultTheme="system"`, `enableSystem` prop)
- **Transition:** `disableTransitionOnChange` on ThemeProvider — no animated transition, instant switch
- **Accessibility:** `<span className="sr-only">Toggle theme</span>` inside button

### Hydration Safety
- `suppressHydrationWarning` on `<html>` element — required, not optional
- Source: RESEARCH.md Pitfall 1 + CONTEXT.md architecture decisions

### Static Export
- `output: "export"` in `next.config.ts` — set before first build
- No `<Image>` components in Phase 1 (avoids static export image optimization conflict)
- Source: RESEARCH.md Pattern 5 + STATE.md decisions
